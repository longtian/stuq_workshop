## 课程简介
本次课程主要介绍了 JavaScript 应用程序性能分析的原理和操作步骤。适合想了解如何分析 JavaScript 应用程序性能的工程师。

## 课程背景介绍：
JavaScript 应用程序分析主要有三种手段 CPU Profile, HeapSnapshot, 和 GC 日志。
